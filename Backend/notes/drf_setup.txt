What is DRF and Why Do We Need It?

Django by default is built for:

HTML templates

Server-side rendering

But we are building:

React frontend + Django backend (API based)

So instead of returning HTML pages, we return:

{
  "title": "Learn DRF",
  "completed": false
}

DRF helps us:

Build REST APIs easily

Serialize models into JSON

Handle authentication

Add permissions

Validate input

Build scalable APIs

Without DRF, building APIs manually in Django is painful.
